# Note Taking Web App

This is a note taking app made with React, TypeScript, and browser `localStorage`.<br>
https://client-five-gray-80.vercel.app/

- Every note has a subject and body. You can write notes and save them directly in your browser.
- You can edit and delete notes at any time
- Each note has an updated and creation date
- Client side validation for forms
- No sign-in or user accounts required
- No backend or database required
